## Introduction

This repository contains various shell scripts to automatize some boring jobs.

### wp.sh

Install [WordPress](https://it.wordpress.org/) in Italian + MySQL 8.0 + PHP on Ubuntu Server.

### mysql_server.sh

Install MySQL 8.0 on Ubuntu Server.
